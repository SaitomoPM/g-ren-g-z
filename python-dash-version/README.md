# GÖREN Performans Takip Sistemi - Python Dash

T.C. Sağlık Bakanlığı · Ankara İl Sağlık Müdürlüğü

## Kurulum

```bash
cd python-dash-version
pip install -r requirements.txt
python app.py
```

Tarayıcıda **http://127.0.0.1:8050** adresine gidin.

## Özellikler

- 3 katmanlı sticky header (kırmızı şerit, marka çubuğu, bağlam çubuğu)
- Filtre kartı (hastane + gösterge seçimi)
- GÖREN / Rapor Kartı sekmeleri
- 4 KPI scorecard
- Dönem karşılaştırması
- Aylık değişim + akıllı öncelik insight kartları
- Trend analizi ve tahmin grafiği
- Dağılım analizi (yatay bar)
- Kritik parametre trendi (çizgi grafik)
- Anomali tespiti
- Tahmine dayalı uyarılar
- Hastane karşılaştırması (gruplu bar grafik)
- Footer
